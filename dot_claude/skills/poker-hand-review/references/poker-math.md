# Poker Math Reference

ハンドレビューで使用する数学的計算の一覧。

## Pot Odds（ポットオッズ）

コールに必要な最低エクイティを算出する。

```
必要エクイティ = コール額 / (現在のポット + ヴィランのベット + コール額)
```

### 一般的なベットサイズとポットオッズ

| ベットサイズ（対ポット比） | 必要エクイティ |
|---------------------------|---------------|
| 25% pot | 16.7% |
| 33% pot | 20.0% |
| 50% pot | 25.0% |
| 66% pot | 28.4% |
| 75% pot | 30.0% |
| 100% pot | 33.3% |
| 150% pot | 37.5% |
| 200% pot | 40.0% |

## Implied Odds（インプライドオッズ）

将来のストリートで獲得可能な追加チップを考慮する。

```
実効オッズ = コール額 / (現在のポット + ヴィランのベット + コール額 + 将来の獲得見込み)
```

**インプライドオッズが高い条件:**
- スタックが深い（SPR > 3）
- ナッツ寄りのドロー（セットマイン、フラッシュドロー）
- ヴィランが強いハンドをペイオフしやすい

**インプライドオッズが低い条件:**
- スタックが浅い（SPR < 2）
- リバースインプライドオッズのリスクが高い（ドミネートされやすいハンド）
- ヴィランのレンジがキャップされている

## Stack-to-Pot Ratio（SPR）

フロップ時点でのスタックとポットの比率。意思決定の指針になる。

```
SPR = 残りの有効スタック / フロップ時のポット
```

| SPR | 戦略的含意 |
|-----|-----------|
| < 2 | コミットゾーン。トップペア以上でオールイン可能 |
| 2-4 | セミコミット。オーバーペア+で快適にスタックオフ |
| 4-8 | 標準的。ツーペア+でスタックオフ、ドロー込みの判断が重要 |
| 8-13 | ディープ。ナッツ級が必要。ドローの価値が上がる |
| > 13 | ウルトラディープ。セットマイン、スーテッドコネクタの価値が高い |

## Minimum Defense Frequency（MDF）

ベットに対してフォールドしすぎないための最低コンティニュー頻度。

```
MDF = ポットサイズ / (ポットサイズ + ベットサイズ)
```

| ベットサイズ | MDF |
|-------------|-----|
| 25% pot | 80.0% |
| 33% pot | 75.0% |
| 50% pot | 66.7% |
| 66% pot | 60.2% |
| 75% pot | 57.1% |
| 100% pot | 50.0% |
| 150% pot | 40.0% |
| 200% pot | 33.3% |

## Bluff-to-Value Ratio（ブラフ対バリュー比率）

バランスの取れたベッティングレンジを構成するための比率。

```
ブラフ比率 = ベットサイズ / (ポット + ベットサイズ + ベットサイズ)
```

リバーでの理論的ブラフ比率:

| ベットサイズ | ブラフ比率 |
|-------------|----------|
| 33% pot | 20% |
| 50% pot | 25% |
| 75% pot | 30% |
| 100% pot | 33% |
| 150% pot | 37.5% |

## EV Calculation（期待値計算）

```
EV = (勝率 × 獲得額) - (敗率 × 損失額)
```

コールのEV:
```
EV(call) = (エクイティ × ポット合計) - (1 - エクイティ) × コール額
```

ブラフベットのEV:
```
EV(bluff) = (フォールドエクイティ × 現ポット) - (1 - フォールドエクイティ) × ベット額
```

## Fold Equity（フォールドエクイティ）

```
フォールドエクイティ = ヴィランのフォールド頻度 × (ポット + デッドマネー)
```

セミブラフの総合EV:
```
EV = フォールドエクイティ + (1 - フォールド頻度) × (エクイティ × ポット全体 - (1 - エクイティ) × リスク額)
```

## Equity Realization（エクイティリアリゼーション）

生のエクイティが実際にどの程度実現されるかの係数。

**エクイティリアリゼーションが高い要因:**
- ポジションがある（IP）
- ナッツアドバンテージがある
- ポラライズされたレンジ
- スキルエッジ

**エクイティリアリゼーションが低い要因:**
- ポジションがない（OOP）
- キャップされたレンジ
- ドミネートされやすいハンド
- マルチウェイポット
